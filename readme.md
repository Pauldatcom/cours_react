# Cours React – Exercices de base

Ce dossier contient plusieurs exercices pratiques pour apprendre les **fondamentaux de React**.

## Contenu

- **useState** → gestion des états simples (compteurs, formulaires, etc.)
- **useEffect** → effets de bord, timers, fetch de données, simulations de chargement
- **useReducer** → gestion d’un état complexe (ex: panier, todo list)
- **props & state** → communication entre composants
- **mini-projets** → ex: tableau de températures, compteur interactif, gestion de panier

## Environnement

Tous les exercices sont exécutables directement via un fichier HTML avec :
- React & ReactDOM via CDN
- Babel pour JSX
- TailwindCSS pour le style

## Objectif

Pratiquer la logique de React sans configuration complexe (pas de Vite/CRA), comprendre les hooks fondamentaux et préparer les futurs projets en composants modernes.
